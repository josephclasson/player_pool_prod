1) Launch-Day Checklist (Minute by Minute)

T-60 to T-30 minutes: Freeze and backup

  • Announce freeze window: no draft picks and no commissioner bulk actions.
  • Confirm target identifiers: league UUID, league code, and season year.
  • Create Supabase backup/snapshot before cutover.
  • Confirm production env vars are prepared.

T-30 to T-15 minutes: Vercel setup

  • Import/connect the repository in Vercel.
  • Set required production environment variables.
  • Start production deploy.
  • Attach domain and complete DNS updates.

T-15 to T-5 minutes: Supabase auth URL check

  • Add redirect URL: https://yourdomain.com/auth/confirm
  • Add redirect URL: https://yourdomain.com/join

T-5 to T+15 minutes: Validation and release

  • Confirm production loads the same league and draft results.
  • Confirm leaderboard and scores load without errors.
  • Confirm commissioner endpoints reject unauthenticated requests.
  • Confirm invite flow works end-to-end.
  • Check runtime logs for env or auth errors.
  • Lift freeze and announce go-live.

4) SQL Verification (Run Pre and Post Deploy)

These checks validate that league identity and draft data remain unchanged through deployment.

-- League identity select id, code, season_year, name, created_at from leagues where id = '142ae951-b6c7-42d2-8401-1c6f01f3b1b7' or (code = 'CHALK26' and season_year = 2026) order by season_year desc; -- Members and roles select league_id, user_id, role, is_autodraft, created_at from league_members where league_id = '142ae951-b6c7-42d2-8401-1c6f01f3b1b7' order by role, created_at; -- Draft picks count select league_id, count(*) as pick_count, min(created_at) first_pick_at, max(created_at) last_pick_at from draft_picks where league_id = '142ae951-b6c7-42d2-8401-1c6f01f3b1b7' group by league_id; -- Roster occupancy select league_id, count(*) as roster_rows, count(*) filter (where player_id is not null) as filled_slots from league_rosters where league_id = '142ae951-b6c7-42d2-8401-1c6f01f3b1b7' group by league_id; -- Live scoreboard cache freshness select league_id, updated_at from league_live_scoreboard where league_id = '142ae951-b6c7-42d2-8401-1c6f01f3b1b7'; -- Optional latest picks snapshot select * from draft_picks where league_id = '142ae951-b6c7-42d2-8401-1c6f01f3b1b7' order by created_at desc limit 25;
